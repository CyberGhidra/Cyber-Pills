# Wireshark

## How to install wireshark in LINUX
```
sudo add-apt-repository ppa:wireshark-dev/stable
sudo apt update
sudo apt install wireshark
```
## play in linux 
```
sudo wireshark 
password:...
```
OR 
```
sudo wireshark -i enp0s3 -k
password: ...
→ Start capturing packets on interface enp0s3
→ Begin analysis

```
