# CCNA 200-301 — Study Roadmap

---

## Block 1 — Theoretical Foundations (Ch. 1–3)
> **Goal:** Understand how networks work before touching any device.

| Chapter | Topic | Study method |
|---------|-------|--------------|
| 01 | Introduction to TCP/IP Networking | Summaries + flashcards |
| 02 | Fundamentals of Ethernet LANs | Summaries + flashcards |
| 03 | Fundamentals of WANs and IP Routing | Summaries + flashcards |

---

## Block 2 — Cisco Switch Configuration (Ch. 4–10)
> **Goal:** Learn the CLI, then apply everything on a real/simulated switch.

| Chapter | Topic | Study method |
|---------|-------|--------------|
| 04 | Using the Command-Line Interface | CLI practice (Packet Tracer) |
| 05 | Analyzing Ethernet LAN Switching | Lab + verification commands |
| 06 | Configuring Basic Switch Management | Lab configs |
| 07 | Configuring and Verifying Switch Interfaces | Lab configs |
| 08 | Implementing Ethernet VLANs | Lab configs |
| 09 | Spanning Tree Protocol Concepts | Theory + topology diagrams |
| 10 | RSTP and EtherChannel Configuration | Lab configs |

---

## Block 3 — IPv4 Subnetting (Ch. 11–15)
> **Goal:** Master subnetting through timed drills — reading alone is not enough.

| Chapter | Topic | Study method |
|---------|-------|--------------|
| 11 | Perspectives on IPv4 Subnetting | Theory + mental model |
| 12 | Analyzing Classful IPv4 Networks | Exercises |
| 13 | Analyzing Subnet Masks | Timed exercises |
| 14 | Analyzing Existing Subnets | Timed exercises |
| 15 | Subnet Design | Timed exercises |

---

## Block 4 — Routers & OSPF (Ch. 16–24)
> **Goal:** Configure IPv4 static routing and OSPF on Cisco routers.

| Chapter | Topic | Study method |
|---------|-------|--------------|
| 16 | Operating Cisco Routers | CLI practice |
| 17 | Configuring IPv4 Addresses and Static Routes | Lab configs |
| 18 | IP Routing in the LAN | Lab configs |
| 19 | IP Addressing on Hosts | Theory + verification |
| 20 | Troubleshooting IPv4 Routing | Troubleshooting labs |
| 21 | Understanding OSPF Concepts | Theory + diagrams |
| 22 | Implementing Basic OSPF Features | Lab configs |
| 23 | Implementing Optional OSPF Features | Lab configs |
| 24 | OSPF Neighbors and Route Selection | Labs + verification |

---

## Block 5 — IPv6 (Ch. 25–29)
> **Goal:** Apply IPv4 knowledge to IPv6 — similarities make it easier.

| Chapter | Topic | Study method |
|---------|-------|--------------|
| 25 | Fundamentals of IP Version 6 | Theory + IPv4 comparison |
| 26 | IPv6 Addressing and Subnetting | Exercises |
| 27 | Implementing IPv6 Addressing on Routers | Lab configs |
| 28 | Implementing IPv6 Addressing on Hosts | Lab configs |
| 29 | Implementing IPv6 Routing | Lab configs |

---

## Block 6 — Review & Exam (Ch. 30 + Mock Tests)
> **Goal:** Fill gaps, consolidate weak spots, simulate exam conditions.

| Task | Description |
|------|-------------|
| Chapter 30 | Exam updates — check for any last-minute changes |
| Practice tests | Full-length mock exams under timed conditions |
| Weak spots | Revisit any block where score < 80% |
| Flashcard review | Quick pass through all cards before exam day |

16–24 → teoria + lab mix
